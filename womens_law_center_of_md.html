TASK

The original website was designed before mobile usage was prevalent.
Thus, the site needed to be updated so that users can find information on both web and mobile platforms.


(BEFORE)




PROCESS

The first step in our web designing process was creating a sitemap based on the structure of the old website.
In order to improve the site navigation, we decided to create one-page level menu (instead of the previous two-page level).
Once we mapped out the website, we designed unique page templates. Some of the templates we made include blog posts and donation forms.








ACCOMPLISHMENTS

Not only were we successful in making the website mobile-friendly and migrating all the information
from the old website to the new website, but we also accomplished everything in 10 consecutive hours.

Additionally, I added a floating translation button to the website so that diverse clients can easily read all the webpages.
I came up with this feature after talking to the founder about how she wants to attract and retain diverse clients, as well as build better rappot with them.
